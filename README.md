## üõ†Ô∏è Master Patterns Summary

### **1. Two Pointers**

Used for searching or manipulating arrays by tracking two positions.

* **Key Insight:** Reduces $O(N^2)$ nested loops to $O(N)$ by moving pointers toward each other or across different sequences.

### **2. Sliding Window**

A subset of two pointers used specifically for **contiguous** subarrays or substrings.

* **Key Insight:** Avoids redundant calculations by "sliding" a window and updating the state (sum, count) as elements enter or leave.

### **3. Prefix Sum**

A pre-computation technique for cumulative totals.

* **Key Insight:** Allows for $O(1)$ range sum queries by storing the "sum-so-far" at each index.

### **4. Hashing**

Using Hash Maps and Sets for efficient data retrieval.

* **Key Insight:** Exchanges space for time to achieve  average-case complexity for insertions and lookups.
